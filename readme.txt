How to use:

1) Install OpenVPN (https://openvpn.net/index.php/open-source/downloads.html). This is required for the TAP device driver.

2) Create config file. An example can be found in peervpn.conf.

3) Run cmd.exe with administrator privileges.

4) Change to the directory where PeerVPN and the config file is located.

5) Run: peervpn.exe your_config_file.conf

6) Enjoy!
